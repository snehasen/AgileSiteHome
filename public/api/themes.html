View

There’re two methods for views: render and renderSync. These two methods are same but the former is synchronous and the letter asynchronous. So I only take render for example.

You can pass options in render method. render will try to process the template with the corresponding renderer and load helpers. When rendering done, it will try to find whether the layout exists. If layout is false or does not exist, it will return the result directly.
